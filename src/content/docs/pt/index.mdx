O Projeto App Fair é uma organização de caridade sem fins lucrativos dedicada à criação de aplicativos de software móvel para o bem público. O projeto foi fundado em 2022 como um nexo mundial para a construção, distribuição e manutenção de aplicativos gratuitos.

Os aplicativos da App Fair não custam nada e não contêm tecnologia de anúncios embutida ou tecnologias de rastreamento não consensuais. Eles não tentam monetizar a atenção do usuário de forma alguma. Eles são criados a partir de software gratuito e de código aberto e são financiados exclusivamente por doações e subsídios.
